## 📚How to deploy

- Step 1: Deploy the application:
```bash
kubectl apply -f https://raw.githubusercontent.com/EliasDeHondt/ChessK8s/refs/heads/main/Kubernetes/ChessK8s.yaml
```

- Step 2: Get the ingress:
```bash
kubectl get ingress
```
